# Summary

This repository contains resources for my paper, Do people think fast or slow when training AI, which is published in the Proceedings of the ACM Conference on Fairness, Accountability, and Transparency (FAccT), 2025. It includes:

Data: Anonymized data used in the paper.
Analysis: R scripts for all experiments, including helper function and attention check to clean and analze the data.
Experiments: Zip files for each experiment, coded using jsPsych (JavaScript, HTML, and CSS).
A link to the OSF repository can be found here: https://osf.io/2caqy/
